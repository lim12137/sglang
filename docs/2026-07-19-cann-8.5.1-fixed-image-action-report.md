## 目标（已实现于本地仓库）

1. **SGLang 基础镜像**：每次构建拉取 **最新** CANN 8.5 上游（默认 `lmsysorg/sglang:main-cann8.5.0-910b`）
2. **ASR 匹配镜像**：运行时挂载指定模型（Nano / VAD / CAM++），**标签后缀 `-asr`**
3. **按构建时间打 tag**：`YYYYMMDD-HHMMSS` / `YYYYMMDD-HHMMSS-asr`，并附带滚动指针 `latest` / `latest-asr`
4. GitHub Actions 推送 **GHCR + 阿里云 ACR**

> 说明：Docker 镜像 tag **不允许 `+`**，因此用 `-asr` 实现与“+asr”相同的区分语义。
